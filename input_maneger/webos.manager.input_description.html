Менеджер входов предназначен для назначения приложений на вход. В текущей версии есть возможность управлять настройками входа для приложения ott play (ID: com.ottplay.app) и его модификаций DRM-play (ID:com.ott.drm) и Ott-FOSS (ID:com.ottplay.foss)
